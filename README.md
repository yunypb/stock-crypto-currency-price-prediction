# Программный комплекс для прогнозирования стоимости и доходности акций и криптовалют

Выпускная квалификационная работа на тему "Предсказание стоимости акций\криптовалют". Работу выполнил Мосунов Данил Витальевич. Разработана модульная система на Python, реализующая полный цикл: от загрузки исторических данных до обучения моделей, расчёта метрик точности и бэктеста торговых стратегий с учётом комиссий.

## Основные возможности

- Загрузка дневных цен закрытия акций (SBER, YNDX) из MOEX и криптовалют (BTC, ETH) из Yahoo Finance / CoinMarketCap.
- Предобработка: заполнение пропусков, расчёт доходностей, генерация широкого спектра признаков (лаги, скользящие средние, волатильность, моментум, RSI, полосы Боллинджера, календарные признаки).
- **Семь прогнозных моделей**, разделённых на три класса:
  - **Статистические**: SMA (простое скользящее среднее), ETS (экспоненциальное сглаживание Хольта–Винтерса), SARIMA.
  - **Ансамблевые**: RandomForest, CatBoost.
  - **Нейросетевые**: LSTM, 1D‑CNN (реализованы на PyTorch).
- Два режима прогнозирования: **абсолютные цены** и **дневные доходности**.
- Walk‑forward валидация с расширяющимся окном (без заглядывания в будущее).
- Торговый бэктест с настраиваемым порогом сигнала и учётом транзакционных издержек.
- Метрики точности: RMSE, MAE, MAPE (для цен); финансовые метрики: итоговый капитал, коэффициент Шарпа, максимальная просадка, количество и доля прибыльных сделок.
- Автоматическое сохранение результатов в Excel и интерактивных HTML‑графиках (Plotly).
